# 赛事方测试方法

本目录提供模拟赛事方批量测试 Docker 镜像的脚本。

## 1. 准备镜像 tar 文件

在项目根目录构建并导出镜像：

```bash
docker buildx build --platform linux/amd64 --build-arg IMAGE_NAME=nvidia/cuda -t bdc2026 .
docker save -o team_name.tar bdc2026:latest
```

将导出的 tar 文件放入：

```text
test/tars/
```

## 2. 配置待测文件列表

编辑：

```text
test/tar_files_list.txt
```

每行写入一个 tar 文件名，例如：

```text
team_name.tar
```

## 3. 运行测试脚本

Linux:

```bash
python test/test.py
```

Windows:

```powershell
python test/test_windows.py
```

## 4. 查看结果

测试完成后，结果会写入：

```text
test/result.csv
```

示例：

```csv
Team Name,Final Score
team_name,0.018867553640330992
```

如果某个镜像无法运行、没有生成合法的 `result.csv`，对应队伍会得到失败分数。
